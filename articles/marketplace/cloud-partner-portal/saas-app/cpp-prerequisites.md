# <a name="saas-application-publishing-prerequisites"></a>SaaS アプリケーションの発行の前提条件

この記事では、Azure Marketplace で SaaS アプリケーション (アプリ) オファーを発行するための前提条件について説明します。  まだお読みになっていない場合は、「[SaaS アプリケーションのオファー発行ガイド](../../marketplace-saas-applications-technical-publishing-guide.md)」をご確認ください。 


## <a name="publishing-prerequisites"></a>発行の前提条件

新しい SaaS アプリ オファーを発行するには、次の前提条件を満たす必要があります。
- [Cloud パートナー ポータル](https://cloudpartner.azure.com/)にアクセスできること。  Cloud パートナー ポータルでは、ロールベースのアクセスが提供され、複数の人が協力してオファーを発行できます。 
- パブリッシャー アカウントを代表してオファーを発行する前に、所有者ロールを持つ人の 1 人が、[使用条件](https://azure.microsoft.com/support/legal/website-terms-of-use/)、[Microsoft のプライバシーに関する声明](https://www.microsoft.com/privacystatement/default.aspx)、[Microsoft Azure 認定プログラム契約](https://azure.microsoft.com/support/legal/marketplace/certified-program-agreement/)を遵守することに同意する必要があります。
- 英語以外のすべてのコンテンツには、英語版が同梱される必要があります。 このコンテンツには、ネットショップのテキスト、ドキュメント、スクリーンショット、利用規約、およびプライバシー ポリシーが含まれます。 英語以外のアプリケーションへの役に立つリンクの URL を指定してもかまいません。 
- メタデータの使用準備ができていること。 次の一覧 (一部) は、このメタデータの例を示すものです。
  - タイトル
  - (HTML 形式) での説明
  - PNG 形式で次の固定画像サイズのロゴ画像:40 x 40 ピクセル、90 x 90 ピクセル、115 x 115 ピクセル、255 x 115 ピクセル。
  - "*利用規約*" と "*プライバシー ポリシー*" に関する声明
  - アプリケーションのドキュメント
  - サポートの連絡先


## <a name="next-steps"></a>次の手順
