# <a name="commercial-marketplace-analytics-common-questions"></a>コマーシャル マーケットプレース分析の一般的な質問

この記事では、パートナー センターの分析メッセージに関してよく寄せられる質問について説明します。

## <a name="common-questions"></a>一般的な質問

### <a name="why-am-i-unable-to-view-my-analytics-data-in-partner-center"></a>パートナー センターで分析データを表示できないのはなぜですか?

分析ページにアクセスすると、次のメッセージが表示されます。

[![あなたのサービスにはデータがありません。](./media/analytics-faq-no-data.png)](./media/analytics-faq-no-data.png#lightbox)

このメッセージが表示される理由:

- コマーシャル マーケットプレースで、あなたが公開したサービスに対する購入が現在ありません。 これは、あなたのサービスがコマーシャル マーケットプレースで販売されており、顧客が製品表示ページを閲覧していますが、顧客がそのサービスを購入およびデプロイするための操作をまだ行っていないことを示します。
- あなたのサービスが公開処理中であり、まだ販売されていない可能性もあります。 顧客が購入できるのは、販売が始まったサービスだけです。 サービスの状態を確認するには、[分析ダッシュボード](https://partner.microsoft.com/dashboard/commercial-marketplace/analytics/summary)の **[概要]** ページを確認します。 詳細については、「[コマーシャル マーケットプレース分析の [概要] ダッシュボード](summary-dashboard.md)」を参照してください。
- あなたのサービスが **[Contact Me]\(連絡する\)** としてリストアップされている可能性があります。[連絡する] はリストのみのサービスであり、顧客はコマーシャル マーケットプレースで購入できません。 そのようなサービスが潜在顧客を生み、その情報が提供されることがありますが、購入できないため、注文されません。 サービスのリストアップの種類を確認するには、サービスの設定ページに移動します。

### <a name="i-know-i-have-analytics-data-so-why-does-the-no-analytics-available-message-appear"></a>分析データがあるのに [No Analytics Available]\(分析を利用できません\) というメッセージが表示されるのはなぜですか?

[![あなたのサービスにはデータがありません。](./media/analytics-faq-no-data.png)](./media/analytics-faq-no-data.png#lightbox)

このメッセージが表示される理由:

- このメッセージが表示された場合、分析データはあるが、選択した計算期間にはデータがないことを意味します。 2010 年以降の特定のデータを表示するには、別の計算期間または月の範囲を選択します。
- フィルター選択でさまざまなディメンションから 1 つ以上のカテゴリを選択した場合、その選択に対して分析データがない可能性があります。 フィルターをリセットするか、フィルターで別の選択を行ってみてください。
