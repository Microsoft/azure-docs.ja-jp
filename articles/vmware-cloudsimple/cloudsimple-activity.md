# <a name="activity-management-overview"></a>アクティビティ管理の概要

AVS では、AVS プライベート クラウド環境の機能に影響を与える可能性がある、あらゆるアクティビティが追跡記録されます。 アクティビティには、アラート、イベント、タスク、監査アクティビティなどがあります。 [[アクティビティ]](monitor-activity.md) ページには、現在のすべてのアクティビティがまとめられており、詳細情報をドリルダウンできます。

## <a name="events"></a>events

イベントでは、AVS ポータルでのユーザーとシステムのアクティビティが追跡されます。 イベントは特定のリソースに関連するアクティビティと影響の重大度を示します。  AVS ポータルからイベントを表示できます。

## <a name="alerts"></a>警告

アラートは、AVS 環境での重要なアクティビティの通知です。 請求やユーザー アクセスに影響するイベントはアラートとして表示されます。 AVS ポータルからアラートを確認できます。

## <a name="tasks"></a>処理手順

タスクでは、完了するまでに 30 秒以上かかる、あらゆるユーザー操作が追跡されます。 AVS ポータルからタスクの進行状況を監視できます。 タスクが完了すると、完了にかかった合計時間が情報に含まれます。

## <a name="audit"></a>Audit

監査ログでは、ユーザーの操作が追跡されます。 監査ログには、ユーザーが操作に指定したパラメーターが含まれます。 監査ログを使用して、すべてのユーザーのユーザー アクティビティを監視できます。
