# <a name="azure-vmware-solution-by-cloudsimple"></a>Azure VMware Solution by CloudSimple

Azure VMware Solution by CloudSimple へようこそ。 CloudSimple を使用する方法と、本サービスでソリューションを有効にする方法について詳しくご確認いただけます。 記事を検索するか、目次を参照してください。

- [概要](cloudsimple-vmware-solutions-overview.md)を表示し、Azure VMware Solution by CloudSimple について詳しく確認します。
- 「[CloudSimple プライベート クラウドの概要](cloudsimple-private-cloud.md)」を選択し、CloudSimple プライベート クラウド ソリューションについて詳しく確認します。
- [ノード](cloudsimple-node.md)を選択し、CloudSimple ノードについて詳しく確認します。
- [初期化と容量のプロビジョニング](quickstart-create-cloudsimple-service.md)を選択し、Azure VMware Solution by CloudSimple の使用を開始します。

| 概念の概要     | Resource                    | 目的                                                                       | 管理         |
|--------------------------|-----------------------------|-------------------------------------------------------------------------------|--------------------|
| **サービスの作成**     | CloudSimple サービス         | CloudSimple サービス エッジ ネットワークを構成する                                    | Azure ポータル       |
|                          | CloudSimple ノード            | CloudSimple サービスで使用する容量をプロビジョニングする                          | Azure ポータル       |
| **サービスの使用**  | プライベート クラウド               | VMware 環境を作成する                                                    | CloudSimple ポータル |
|                          | VLAN/サブネット                 | VMware を使用するために土台となるネットワークを作成する                               | CloudSimple ポータル |
|                          | ファイアウォール テーブル              | 土台となるネットワークで東西南北のセキュリティ規則を構成する | CloudSimple ポータル |
|                          | パブリック IP アドレス           | VMware ワークロード用にパブリックに公開されているアドレスを取得する                           | CloudSimple ポータル |
|                          | Azure 接続            | Azure でオンプレミス接続または仮想ネットワークへの接続を確立する         | CloudSimple ポータル |
| **統合された管理**   | CloudSimple 仮想マシン | Azure portal で VMware 仮想マシンンを管理する                                | Azure ポータル       |
