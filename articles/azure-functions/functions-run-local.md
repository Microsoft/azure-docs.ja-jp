# <a name="how-to-code-and-test-azure-functions-locally"></a>Azure 関数をローカルでコーディングしてテストする方法
> [!NOTE]
> ローカルでの Azure Functions の開発およびツールのエクスペリエンスは現在プレビューの段階であり、最終リリースでは大幅に改良される予定です。 Azure Functions ホストをローカルで実行する最も簡単な方法は、[Azure Functions CLI](https://go.microsoft.com/fwlink/?linkid=832752) を使用して npm からインストールすることです。 現時点では、Windows のみがサポートされています。 完全なドキュメントは間もなく公開されます。 Azure Functions CLI に関するフィードバックの提供またはバグの登録を行うには、[azure-webjobs-sdk-script](https://github.com/Azure/azure-webjobs-sdk-script) GitHub リポジトリに問題を登録してください。
