# <a name="managing-your-academic-grant"></a>教育機関の許可を管理する

Azure Education Hub の主なランディング ページは、[概要] ページです。 このページには、確立したクラスの数や、これらのクラスの割り当てられて使用された総実行クレジットなど、教育機関の許可についてのすべての関連情報が含まれています。 また、Education Hub によって提供されるその他の利点へのショートカットも表示されます。ソフトウェア、学習、およびクイックスタート テンプレートです。

## <a name="overview-page"></a>[概要] ページ
:::image type="content" source="media/hub-overview-page/overview.png" alt-text="サインインのエラー メッセージ。" border="false":::

1. **[コース]** は、インストラクターのアカウントのアクティブなコースの合計数を示します。
1. **[ラボ]** は、学生に配布されたアクティブなラボの合計数を示します。
1. **[必要なアクション]** には、ラボへの招待の受け入れなど、完了する必要があるすべてのアクションが一覧表示されます。
1. **[ソフトウェア]** には、教師としてダウンロードできる無料のソフトウェアが一覧表示されます。
1. **[学習]** は、Microsoft Learn を通じて利用可能な無料の Azure ラーニング パスにリンクしています。
1. **[クイックスタート テンプレート]** には、一般的なタスクのためのデプロイを高速化および簡略化するのに役立つ Azure テンプレートが含まれています。
