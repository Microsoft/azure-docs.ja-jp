## <a name="data-box-limits"></a>Data Box の制限

- Data Box は、最大で 5 億個のファイルを格納できます。
- Data Box は、クラウドで最大 512 個のコンテナーまたは共有をサポートしています。 ユーザー共有内の最上位レベルのディレクトリは、クラウド内のコンテナーまたは Azure ファイル共有になります。
