# <a name="storsimple-8000-series-update-2-release-notes"></a>StorSimple 8000 シリーズ Update 2 リリース ノート

## <a name="overview"></a>概要
次のリリース ノートでは、StorSimple 8000 シリーズ Update 2 の新機能について説明し、未解決の重要な問題を示します。 また、このリリースに含まれる StorSimple のソフトウェア、ドライバー、およびディスク ファームウェアの更新プログラムの一覧も含まれています。 

Update 2 は、Release (GA) または Update 0.1 から Update 1.2 までを実行しているすべての StorSimple デバイスに適用できます。 Update 2 に関連付けられているデバイスのバージョンは、6.3.9600.17673 です。

StorSimple ソリューションの更新プログラムをデプロイする前に、リリース ノートに含まれる情報を確認してください。

> [!IMPORTANT]
> * この更新プログラムをインストールするには、(Windows の更新プログラムを含めて) 約 4 ～ 7 時間かかります。 
> * Update 2 には、ソフトウェア、LSI ドライバー、および SSD ファームウェアの更新が含まれます。
> * 新しいリリースについては、更新プログラムの公開を段階的に行うため、更新プログラムが即座に表示されない場合があります。 これらの更新プログラムは間もなく利用可能になるため、数日待ってから、もう一度、更新プログラムの有無を確認してください。
> 
> 

## <a name="whats-new-in-update-2"></a>Update 2 の新機能
Update 2 では、次の新機能が導入されています。

* **ローカル固定ボリューム** - StorSimple 8000 シリーズの以前のリリースでは、データ ブロックは使用量に基づいてクラウドに階層化されていました。 それらのブロックがローカルに保持されることを保証する方法はありませんでした。 Update 2 では、ボリュームを作成するときに、ローカル固定としてボリュームを指定することができます。そのボリュームのプライマリ データは、クラウドに階層化されません。 ローカル固定ボリュームのスナップショットは、引き続きバックアップ用にクラウドにコピーされます。そのため、データのモビリティとディザスター リカバリーの目的でクラウドを利用することができます。 また、ボリュームの種類を変更することができます (階層化ボリュームからローカル固定ボリュームへの変換、およびローカル固定ボリュームから階層化ボリュームへの変換)。 
* **StorSimple 仮想デバイスの機能強化** - 以前は、StorSimple 8000 シリーズはディザスター リカバリーや開発/テストのソリューションとして仮想デバイスをプロビジョニングしていました。 仮想デバイスのモデルは 1 つだけでした (モデル 1100)。 Update 2 では、次の 2 つの仮想デバイス モデルが導入されています。 
  
  * 8010 (旧称: 1100) - 変更はありません。30 TB の容量があり、Azure Standard Storage を使用します。
  * 8020 - 64 TB の容量があり、パフォーマンス向上のために Azure Premium Storage を使用します。
    
    両方の仮想デバイス モデル (8010/8020) に対して 1 つの VHD があります。 仮想デバイスを初めて起動すると、プラットフォームのパラメーターが検出され、適切なモデル バージョンが適用されます。
* **ネットワークの機能強化** - Update 2 には、次のようなネットワークの機能強化が含まれています。
  
  * NIC で障害が発生した場合にフェールオーバーを実行できるように、クラウドで複数の NIC を有効にできます。
  * クラウド対応のブロック用に固定メトリックを使用してルーティングが強化されました。
  * フェールオーバーの実行前に、障害の発生したリソースのオンラインでの再試行が行われます。
  * サービスの障害に対して新しいアラートが追加されました。
* **更新の機能強化** - Update 1.2 以前では、StorSimple 8000 シリーズの更新には、2 つのチャネル (クラスタリングや iSCSI などには Windows Update、バイナリとファームウェアには Microsoft Update) が使用されていました。
    Update 2 では、すべての更新パッケージで Microsoft Update が使用されます。 これにより、パッチ適用やフェールオーバー実行にかかる時間が短縮されます。 
* **ファームウェアの更新プログラム** - 次のファームウェア更新プログラムが含まれています。
  
  * LSI: lsi_sas2.sys 製品バージョン 2.00.72.10
  * SSD のみ (HDD の更新なし): XMGG、XGEG、KZ50、F6C2、および VR08
* **プロアクティブ サポート** - Update 2 では、Microsoft はデバイスから追加の診断情報を取得します。 Microsoft の運用チームが問題のあるデバイスを識別するときに、デバイスから情報を収集して問題を診断する能力が向上しました。 **Update 2 を受け入れると、このプロアクティブ サポートの提供に同意することになります**。

## <a name="controller-and-firmware-updates-in-update-2"></a>Update 2 のコントローラーとファームウェアの更新プログラム
このリリースでは、ドライバーとデバイス上のディスク ファームウェアが更新されます。

* LSI ファームウェアの更新プログラムの詳細については、Microsoft サポート技術情報 3121900 を参照してください。 
* ディスクのファームウェア更新プログラムの詳細については、Microsoft サポート技術情報 3121899 を参照してください。

## <a name="virtual-device-updates-in-update-2"></a>Update 2 での仮想デバイスの更新プログラム
この更新プログラムは、仮想デバイスには適用できません。 新しい仮想デバイスを作成する必要があります。
