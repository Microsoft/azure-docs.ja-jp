# <a name="register-your-azure-digital-twins-app-with-azure-active-directory-legacy"></a>Azure Active Directory レガシに Azure Digital Twins アプリを登録する

この記事では、古い方法または従来の方法を使用して、サンプル アプリケーションを Azure Active Directory (Azure AD) に登録し、これが Digital Twins インスタンスにアクセスできるようにする方法を示します。 新しい Azure AD アプリ登録がご使用のセットアップで機能しない場合、この方法を試すことをお勧めします。

[!INCLUDE [Digital Twins legacy AAD](../../includes/digital-twins-permissions-legacy.md)]

## <a name="next-steps"></a>次の手順

アプリが AAD に登録された後、そのアプリは Digital Twins インスタンスに接続でき、お客様のシナリオをサポートできます。 次の手順について詳しくは、[クイックスタート](quickstart-view-occupancy-dotnet.md#build-application)または[チュートリアル](tutorial-facilities-setup.md#configure-the-digital-twins-sample)をご覧ください。
