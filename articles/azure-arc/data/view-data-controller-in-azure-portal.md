# <a name="view-azure-arc-data-controller-resource-in-azure-portal"></a>Azure portal で Azure Arc データ コントローラー リソースを表示する

[Azure へのメトリックまたはログの最初のアップロード](upload-metrics-and-logs-to-azure-monitor.md)を実行するか、または[使用状況をアップロード](view-billing-data-in-azure.md)したら、Azure portal で Azure 内に作成された Azure Arc データ コントローラー リソースを表示できるようになります。

特殊な URL [https://portal.azure.com](https://portal.azure.com) を使用して Azure portal を開きます。

検索バーの名前でデータ コントローラーを検索し、それをクリックします。
