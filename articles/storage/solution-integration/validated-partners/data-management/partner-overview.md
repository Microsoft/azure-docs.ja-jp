# <a name="azure-storage-data-governance-management-and-migration-partners"></a>Azure Storage のデータのガバナンス、管理、移行パートナー

この記事では、データ管理機能全体を向上させることができる、Azure Storage に統合された Microsoft パートナー企業について説明します。 これらのパートナー ソリューションを使用して、ストレージの評価とレポート、プラットフォームに依存しない移行、レプリケーション、クラウドを使った階層化、またはデータ ガバナンスをサポートできます。
