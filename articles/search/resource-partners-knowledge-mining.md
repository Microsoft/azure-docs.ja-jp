# <a name="partner-spotlight"></a>パートナーの紹介

Azure Cognitive Search が含まれているエンドツーエンドのソリューションを構築する Microsoft パートナーの専門家による支援を受けることができます。 次のパートナーは、さまざまなビジネスおよび技術的なシナリオでフルテキスト検索と AI エンリッチメントを活用するための深い経験を持っています。
