# <a name="azure-synapse-analytics-launch-partners"></a>Azure Synapse Analytics のローンチ パートナー

Azure Synapse Analytics のパートナーは、さまざまな業界向けの統合ツールとデリバリーに関する高度な専門知識を備えており、クラウド スケール分析を使用してお客様のデジタル変革の促進をサポートします。 以下は、社会的増幅やさまざまなワークショップ、各種プランを通じて 5 月 19 日のパブリック プレビューのローンチを支援しているパートナーです。その各種オファリングをご覧ください。
