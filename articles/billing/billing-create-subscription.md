# <a name="create-an-additional-subscription-in-the-azure-portal"></a>Azure portal での追加サブスクリプションの作成

Azure アカウントの追加サブスクリプションを作成できます。 サブスクリプションの制限、請求およびセキュリティの環境の分離、またはコンプライアンス上の理由によるデータの分離などを避けるため、追加でサブスクリプションが必要な場合があります。

組織のエンタープライズ契約の下で Azure サブスクリプションを作成する場合は、組織アカウントの所有者ロールが必要です。

Azure サブスクリプションをまだお持ちでない場合は、[無料のアカウント](https://azure.microsoft.com/free)を作成できます。

## <a name="create-an-additional-azure-subscription"></a>追加 Azure サブスクリプションの作成

1. [Azure Portal](https://portal.azure.com) にサインインします。
1. **サブスクリプション**を検索します。

   ![サブスクリプション用のポータルでの検索を示すスクリーン ショット](./media/billing-create-subscription/billing-search-subscription-portal.png)

1. **[追加]** を選択します。

   ![サブスクリプション ビューの追加ボタンを示すスクリーン ショット](./media/billing-create-subscription/subscription-add.png)

追加サブスクリプションをプログラムで作成することもできます。 詳細については、[Azure Enterprise サブスクリプションをプログラムで作成する](../azure-resource-manager/programmatically-create-subscription.md) を参照してください。
