# <a name="view-the-tax-documents-for-your-microsoft-customer-agreement"></a>Microsoft 顧客契約に関する税務書類を表示する

この記事では、Microsoft 顧客契約の課金アカウントについて説明します。 [Microsoft 顧客契約にアクセスできるかどうかを確認してください](#check-access-to-a-microsoft-customer-agreement)。

## <a name="view-and-download-tax-documents"></a>税務書類を表示してダウンロードする

税務書類を表示してダウンロードするには、課金プロファイルの所有者、共同作成者、閲覧者、または請求書管理者である必要があります。 Microsoft 顧客契約の課金ロールの詳細については、「[課金プロファイルのロールとタスク](billing-understand-mca-roles.md#billing-profile-roles-and-tasks)」を参照してください。
<!-- TO DO: add link to manage access doc for MCA -->

1. **[コストの管理と請求]** で検索します。
2. 課金プロファイルを選択します。
3. **[請求書]** を選択します。
4. 請求書グリッドで、ダウンロードする税務書類に対応する請求書の行を見つけます。
5. 行の末尾にある省略記号 (`...`) をクリックします。
6. **[税務書類]** を選択します。 請求プロファイルの国によっては、請求書ごとに 1 つ以上の税務書類が表示されることがあります。
