# <a name="drop-external-stream-transact-sql"></a>DROP EXTERNAL STREAM (Transact-SQL)

ストリーミング ジョブを削除します。 

## <a name="syntax"></a>構文

```sql
DROP EXTERNAL STREAM {external_stream_name}  
```
