# <a name="app-service-environment-management-addresses"></a>App Service Environment の管理アドレス

App Service Environment (ASE) は、ユーザーの Azure Virtual Network (VNet) 内のサブネットへの Azure App Service のデプロイです。  ASE は、管理できるように、Azure App Service からアクセスできる必要があります。  この ASE 管理トラフィックでは、ユーザーによって制御されるネットワークを走査します。  Azure App Service の管理サーバーから、ASE に関連付けられているパブリック VIP に送信されます。  ASE ネットワークの依存関係の読み取りの詳細については、「[App Service Environment のネットワークの考慮事項][networking]」を参照してください。  最初に参照できる ASE に関する一般情報については、「[App Service 環境の概要][intro]」を参照してください。

このドキュメントでは、ASE に対する管理トラフィックのソース IP アドレスが一覧表示されます。 これらのアドレスを使用して、着信トラフィックをロックするネットワーク セキュリティ グループを作成することも、必要に応じてそれらをルート テーブルで使用することもできます。  この情報を使用するには、以下を使用する必要があります。

* すべての地域に登録されている IP アドレス
* ASE がデプロイされているリージョンに一致する IP アドレス。

管理の着信トラフィックはこれらの IP アドレスからポート 454 および 455 に送信されます。
