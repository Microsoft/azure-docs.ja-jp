# <a name="migrate-to-azure-app-service"></a>Azure App Service に移行する

[App Service Migration Assistant](https://azure.microsoft.com/services/app-service/migration-assistant/) を使用して、オンプレミスのアプリを Azure App Service に移行できます。 App Service Migration Assistant の目的は、オンプレミスからクラウドにアプリケーションを移行するための無料かつシンプルな高速ソリューションを通じて、クラウドへの移行を簡略化することです。

Azure App Service Migration Assistant を使用すると、次のことが簡単にできます。

- アプリの URL をスキャンして、移行に適した候補であるかどうかを評価します。
- Migration Assistant をダウンロードして、移行を開始します。
- ツールを使用して、アプリの構成設定の準備チェックと全般的な評価を実行します。
- ツールを使用して Azure App Service にアプリまたはサイトを移行します。

[Web アプリを Azure App Service に移行する方法を確認してください](https://www.youtube.com/watch?v=9LBUmkUhmXU)。
