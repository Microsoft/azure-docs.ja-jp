<h1>Azure Data Box</h1>
<p>Azure Data Box ファミリには、信頼性が高く、迅速かつ安価な方法で数テラバイト (TB) のデータを Azure に送信できる、ストレージ容量が異なる製品が用意されています。 Microsoft は、オフラインまたはネットワーク経由のデータ転送を可能にする独自のストレージ デバイスをお客様に届けることで、安全なデータ転送を促進しています。</p>
