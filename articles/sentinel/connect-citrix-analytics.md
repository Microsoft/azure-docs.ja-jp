# <a name="connect-citrix-analytics-security"></a>Citrix Analytics (セキュリティ) の接続 

Citrix Analytics (セキュリティ) と Azure Sentinel の統合により、リスクの高いイベントについて分析されたデータを Citrix Analytics (セキュリティ) から Azure Sentinel にエクスポートできます。 カスタム ダッシュボードを作成したり、Citrix Analytics (セキュリティ) からのデータと一緒に他のソースからのデータを分析したり、Logic Apps を使用してカスタム ワークフローを作成してセキュリティ イベントを監視および軽減したりできます。 



## <a name="prerequisites"></a>前提条件

- Azure Sentinel ワークスペースに対する読み取りおよび書き込みアクセス許可。

- Citrix Cloud での Citrix Security Analytics の権利。


## <a name="connect-azure-sentinel-to-citrix-analytics-security"></a>Azure Sentinel を Citrix Analytics (セキュリティ) に接続する

1. Azure portal で、 **[Azure Sentinel]**  >  **[Data connectors]\(データ コネクタ\)** に移動し、 **[Citrix Analytics (Security)]\(Citrix Analytics (セキュリティ)\)** コネクタを選択します。

2. **[Open connector page]\(コネクタ ページを開く\)** を選択します。

3. **Citrix Analytics (セキュリティ)** ページの指示に従います。
