# <a name="mapping-of-azure-security-benchmark-v2-and-cis-microsoft-azure-foundations-benchmark"></a>Azure セキュリティ ベンチマーク V2 と CIS Microsoft Azure Foundations Benchmark のマッピング

CIS Microsoft Azure Foundations Security Benchmark は、Center for Internet Security が提供する、Azure のセキュアなベースライン構成を確立するためのセキュリティ ガイダンスです。 このベンチマークのスコープは、基本レベルのセキュリティを確立すると同時に Azure Cloud を導入することです。 このベンチマークは、 [Azure セキュリティ ベンチマーク v2](overview.md) と連携しています。

[CIS Microsoft Azure Foundations Benchmark 最新バージョン v1.3.0](https://www.cisecurity.org/benchmark/azure/) と [Azure セキュリティ ベンチマーク v2](overview.md) の間のマッピングを提供しています。 これは、2 つのベンチマークの推奨事項がどう対応しているかを理解するのに役立ちます。 マッピング全体は[こちら](https://github.com/MicrosoftDocs/SecurityBenchmarks/blob/master/Azure%20Security%20Benchmark/2.0/asb_v2_to_cis_microsoft_azure_foundations_benchmark_v1.3.0.xlsx)からダウンロードできます。
