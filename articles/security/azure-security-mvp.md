# <a name="azure-security-mvp-program"></a>Azure セキュリティの MVP プログラム

Microsoft Most Valuable Professional (MVP) とは、他のユーザーを献身的に支援している模範的なコミュニティ リーダーのことを指します。 MVP により、他のユーザーは、マイクロソフトのテクノロジに関する経験を最大限に活用できるようになります。 情熱、実世界の知識、および技術的専門知識を、コミュニティや Microsoft と共有します。

Microsoft Azure では、Azure セキュリティに関する特別な専門知識を持つコミュニティ エキスパートを承認するようになりました。 Microsoft MVP は、Azure セキュリティの貢献エリアにおいて、Microsoft Azure の MVP を取得できます。

![Microsoft MVP](./media/azure-security-mvp/azure-security-mvp-fig1.png)

MVP になるための明確な基準はありません。 その理由の 1 つは、テクノロジとそのライフ サイクルによって基準が変わるからです。 基準の例を次に示します。

- Microsoft Answers、TechNet、MSDN などのオンライン フォーラムに対する候補者の貢献度
- Wiki およびオンラインのコンテンツ
- 会議およびユーザー グループ
- ポッドキャスト、Web サイト、ブログ、およびソーシャル メディア
- 記事および書籍。

Azure セキュリティの専門家ですか? 専門家をご存知ですか? [自分自身または他のユーザーを指名して](https://mvp.microsoft.com/Nomination/nominate-an-mvp) 、今すぐ Azure セキュリティ MVP になりましょう。
