# <a name="azure-policy-built-in-definitions-for-azure-lighthouse"></a>Azure Lighthouse 用の Azure Policy 組み込み定義

このページは、Azure Lighthouse 用の [Azure Policy](../../governance/policy/overview.md) 組み込みポリシー定義のインデックスです。 他のサービス用の Azure Policy 組み込みについては、[Azure Policy 組み込み定義](../../governance/policy/samples/built-in-policies.md)に関するページをご覧ください。
