# <a name="azure-content-delivery-network"></a>Azure Content Delivery Network
ストリーミング メディア、ゲーム ソフトウェア、ファームウェアの更新 (スマート TV や消費家電製品など)、またはモノのインターネット (IoT) エンドポイント (車やセンサーなど) が含まれている Web サイトやモバイル アプリに、21Vianet (Azure China 21Vianet) によって運営されている Microsoft Azure を使用すると、Content Delivery Network (CDN) により、ロード時間を短縮し、帯域幅を節約して、応答性を向上させることができます。

中国でローカル プロバイダーが、CDN サービスを提供します (比較すると、グローバル Azure の [Azure CDN](https://azure.microsoft.com/documentation/articles/cdn-overview/) サービスは、Microsoft、Verizon、および Akamai から選択できます)。現時点では、ChinaCache の Webluker、[ChineNetCenter](http://www.chinanetcenter.com/)、[Fastweb](https://www.fastweb.com.cn/) などの 6 つのローカル CDN プロバイダーが利用できますが、今後さらに増えていきます。

Azure CDN サービスを使用するには: 
- Azure CDN を含むすべての Azure サービスの単一の窓口である [21Vianet からサポートを受けます](/azure/china/china-resources-support)。
- Azure [統合 CDN 管理ポータル](https://www.azure.cn/documentation/articles/cdn-management-portal-how-to-use/) (使用したことがあるかもしれないグローバル Azure CDN ポータルではなく) にログオンします。
