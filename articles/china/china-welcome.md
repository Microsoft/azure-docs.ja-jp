# <a name="welcome-to-azure-china-21vianet"></a>Azure China 21Vianet へようこそ
21Vianet (Azure China 21Vianet) が運営する Microsoft Azure は、中国本土内にあるクラウド サービスの物理的に分離されたインスタンスであり、Beijing 21Vianet Broadband Data Center Co., Ltd の完全子会社である Shanghai Blue Cloud Technology Co., Ltd. ("21Vianet") によって独立して運用され、処理されています。 

このサービスは、同等のサービス レベルで Microsoft グローバル クラウド サービスを構成する、同じ Azure、Microsoft Office 365、Microsoft Power BI テクノロジに基づいています。 中国の Microsoft Azure 向けの契約は、適用できる場合、顧客と 21Vianet との間で署名されます。

政府規制に準拠するように中国で提供される最初の外国のパブリック クラウド サービス プロバイダーとして、[Azure China 21Vianet](/azure/china/china-overview-operations) は、アーキテクチャに構築されたすべてのシステムとアプリケーションに対して中国の規制で必要とされる、[セキュリティ センター](https://www.trustcenter.cn/compliance/)で協議されたワールド クラスのセキュリティを提供します。 Azure には、サービスとしてのインフラストラクチャ (IaaS)、サービスとしてのプラットフォーム (PaaS)、およびサービスとしてのソフトウェア (SaaS) のコア コンポーネントがあります。 これらのコンポーネントには、ネットワーク、ストレージ、データ管理、ID 管理、その他多くのサービスが含まれます。

Azure China 21Vianet では、静止データのレプリケーションや自動スケールなど、グローバル Azure で提供されるものと同じ優れた[サービス](https://www.azure.cn/home/features/products-by-region)がほとんどサポートされます。 既にグローバル Azure サービスを使用している場合でも、中国で運用するには、ご利用のアプリケーションやサービスの一部またはすべてを[再ホスト](/azure/china/china-how-to-rehost)または[リファクター](/azure/china/china-how-to-refactor)する必要があります。

- [開発者向けのガイダンス](/azure/china/china-get-started-developer-guide)

## <a name="operations"></a>操作
中国[電気通信規制](http://www.miit.gov.cn/n1146295/n1146557/n1146619/c4860613/content.html) (中国語) に従って、クラウド サービスのプロバイダー (IaaS および PaaS) は、付加価値電気通信の許可を有する必要があります。 外国投資が 50% 未満の国内登録企業のみがこれらの許可を受ける資格があります。 この規制に準拠するため、中国の Azure サービスは Microsoft からライセンスされたテクノロジに基づいて、21Vianet が運用しています。
