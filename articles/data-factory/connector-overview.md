# <a name="azure-data-factory-connector-overview"></a>Azure Data Factory コネクタの概要

Azure Data Factory は、コピー アクティビティ、データ フロー、ルックアップ アクティビティ、メタデータの取得アクティビティ、および削除アクティビティを使用して、次のデータ ストアをサポートします。 各データ ストアをクリックすると、サポートされている機能の詳細と対応する構成を確認できます。
