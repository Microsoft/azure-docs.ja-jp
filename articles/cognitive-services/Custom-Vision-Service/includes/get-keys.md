## <a name="get-the-training-and-prediction-keys"></a>トレーニング キーと予測キーを取得する

このプロジェクトを使ってサービスと対話するには、有効な一連のサブスクリプション キーが必要です。 無料試用版の一連のキーを取得するには、[Custom Vision Web サイト](https://customvision.ai)にアクセスし、Microsoft アカウントでサインインします。 右上にある __歯車アイコン__ を選択します。 __[アカウント]__ セクションで、 __[Training Key]\(トレーニング キー\)__ 、 __[Prediction Key]\(予測キー\)__ 、および __[Prediction Resource Id]\(予測リソース ID\)__ の各フィールドの値を確認します。 この情報は後で必要になります。
