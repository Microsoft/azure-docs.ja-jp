## <a name="reference-docs"></a>リファレンス ドキュメント

Speech サービスからは 2 つの SDK が提供されます。 最初の SDK は主要な [Speech SDK](../speech-sdk.md) であり、Speech サービスとのやりとりに必要な機能の大部分が提供されます。 2 番目の SDK はデバイスに固有であり、それにふさわしい [Speech Devices SDK](../speech-devices-sdk.md) という名前が付いています。 いずれの SDK もさまざまな言語で利用できます。

### <a name="speech-devices-sdk-reference-docs"></a>Speech Devices SDK リファレンス ドキュメント

[Speech Devices SDK](../speech-devices-sdk.md) は Speech SDK の上位集合であり、特定のデバイスの機能が拡張されています。 Speech Devices SDK をダウンロードするには、最初に[開発キットを選択する](../get-speech-devices-sdk.md#choose-a-development-kit)必要があります。
