このクイックスタートでは、コマンド ラインから Speech CLI を使用して、オーディオ ファイルに録音された音声を認識し、テキストへの文字起こしを生成します。 Speech CLI を使用すると、会話の文字起こしなどの一般的な認識タスクを簡単に実行できます。 Speech CLI では、構成を 1 回行えば、マイクで、またはバッチ スクリプトを使用してファイルから、音声をテキストに対話形式で文字起こしすることができます。
