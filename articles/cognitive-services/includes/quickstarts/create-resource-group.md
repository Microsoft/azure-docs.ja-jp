## <a name="create-a-resource-group"></a>リソース グループを作成する

Cognitive Services リソースを作成する前に、リソースを格納するための Azure リソース グループをアカウントに用意する必要があります。 リソース グループがまだ存在しない場合は、先に進む前に、[Azure portal](https://ms.portal.azure.com/) で作成します。
