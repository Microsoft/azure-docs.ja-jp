# <a name="journal-entity"></a>定期刊行物エンティティ

<sub> *次の属性は、定期刊行物エンティティに固有です。(Ty = '2') </sub>

Name    |説明                            |type       | 操作
------- | ------------------------------------- | --------- | ----------------------------
Id      |エンティティ ID                              |Int64      |等しい
DJN     |定期刊行物の標準化名                |String     |なし
JN      |定期刊行物の表示名                   |String     |等しい
CC      |定期刊行物の引用の総数           |Int32      |なし  
ECC     |定期刊行物の引用の推定総数 |Int32      |なし
