### <a name="add-utterances-from-the-command-line"></a>コマンドラインから発話を追加する

引数を指定せずに add-utterance を呼び出すと、アプリに発話は追加されますが、トレーニングは実行されません。

```console
> python add-utterances-3-6.py
```
