他のユーザーによるアプリの作成が必要な場合は、関連付けられているメール アドレスを Azure portal の作成リソースに追加する必要があります。

ご自身が **Azure サブスクリプションの所有者または管理者**の場合は、リソースに共同作成者を追加できます。

ご自身が **Azure サブスクリプションの所有者または管理者でない**場合は、お使いの Azure アカウントに `Microsoft.Authorization/roleAssignments/write` アクセス許可が必要です。

このロールの割り当てに問題がある場合は、以下を確認してください。

* [Azure でのロールの割り当て](../../../role-based-access-control/role-assignments-portal.md)
 * [Azure でのアクセス制御のトラブルシューティング](../../../role-based-access-control/troubleshooting.md#problems-with-rbac-role-assignments):
