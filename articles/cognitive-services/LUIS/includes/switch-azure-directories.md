自分で作成したアプリや共有されたアプリが表示されない場合は、別の Azure ディレクトリに切り替える必要があることがあります。

1. 画面の右上隅にあるアバターをクリックします。 次に、 **[別の Azure ディレクトリに切り替える]** をクリックします。 
2. 表示されるウィンドウで、共有された LUIS リソースが含まれている Azure ディレクトリを選択します。
