# <a name="azure-powershell-samples-for-azure-cosmos-db---table-api"></a>Azure Cosmos DB 用 Azure PowerShell サンプル - Table API

次の表には、Azure Cosmos DB for Table API の Azure PowerShell スクリプトのサンプルへのリンクが含まれています。

| | |
|---|---|
|[アカウントとテーブルの作成](scripts/powershell/table/ps-table-create.md?toc=%2fpowershell%2fmodule%2ftoc.json)| Azure Cosmos アカウントとテーブルを作成します。 |
|[テーブルの一覧表示または取得](scripts/powershell/table/ps-table-list-get.md?toc=%2fpowershell%2fmodule%2ftoc.json)| テーブルを一覧表示または取得します。 |
|[RU/秒の取得](scripts/powershell/table/ps-table-ru-get.md?toc=%2fpowershell%2fmodule%2ftoc.json)| テーブルの RU/秒を取得します。 |
|[RU/秒の更新](scripts/powershell/table/ps-table-ru-update.md?toc=%2fpowershell%2fmodule%2ftoc.json)| テーブルの RU/秒を更新します。 |
|[リージョンの追加](scripts/powershell/common/ps-account-update.md?toc=%2fpowershell%2fmodule%2ftoc.json)| Azure Cosmos アカウントを取得し、場所の一覧にリージョンを追加します。 |
|[フェールオーバー優先度の変更](scripts/powershell/common/ps-account-failover-priority-update.md?toc=%2fpowershell%2fmodule%2ftoc.json)| 手動フェールオーバー トリガーで、Azure Cosmos アカウントのフェールオーバー優先度を変更します。 |
|[アカウント キーと接続文字列のタスク](scripts/powershell/common/ps-account-keys-connection-strings.md?toc=%2fpowershell%2fmodule%2ftoc.json)| Azure Cosmos アカウントの接続文字列を取得します。 Azure Cosmos アカウントのプライマリ キーとセカンダリ キーを取得します。 アカウント キーを再生成します。 |
