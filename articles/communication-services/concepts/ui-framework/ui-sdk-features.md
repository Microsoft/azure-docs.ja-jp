# <a name="ui-framework-capabilities"></a>UI フレームワークの機能

[!INCLUDE [Private Preview Notice](../../includes/private-preview-include.md)]

Azure Communication Services の UI フレームワークでは、一連の再利用可能なコンポーネントを使用して通信エクスペリエンスを構築できます。 これらのコンポーネントには、UI エクスペリエンスの最も基本的な構成要素である **基本** コンポーネントと、この基本コンポーネントの組み合わせである **複合** コンポーネントの 2 種類があります。

## <a name="ui-framework-composite-components"></a>UI フレームワークの複合コンポーネント

| Composite               | 説明                                               | Web   | Android | iOS   |
|-------------------------|-----------------------------------------------------------|-------|---------|-------|
| グループ通話の複合 | Fluent UI のデザイン アセットを使用した Azure Communication Services 通話用の、軽量な音声およびビデオ発信通話エクスペリエンス。 Azure Communication Services のグループ ID を使用してグループ通話をサポートします。 複合では、Azure Communication Services ID、または Azure で購入した電話番号を使用する PSTN の電話番号を参照することによって、1 対 1 の通話を使用できます。                                    | React |  |  |
| グループ チャットの複合    | Fluent UI のデザイン アセットを使用した Azure Communication Services の軽量なチャット エクスペリエンス。 このエクスペリエンスは、Azure Communication Services スレッドに接続できる単純なチャット クライアントの提供に重点を置いています。 これにより、ユーザーは、入力インジケーターや開封確認メッセージを使用して、メッセージを送信したり、受信メッセージを表示したりできます。 1 対 1 のチャットからグループ チャットのシナリオまで対応します。 単一のチャット スレッドをサポートします。                         | React |  |  |

## <a name="ui-framework-base-components"></a>UI フレームワークの基本コンポーネント

| コンポーネント             | 説明                                                                                                                                                                                                                                                                        | Web   | Android | iOS |
|-----------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------|---------|-----|
| 通話プロバイダー    | 通話用のコア初期化コンポーネント。 必須コンポーネント。このコンポーネントを基盤とする他のコンポーネントを初期化します。 Azure Communication Services のアクセス トークンを使用して、通話クライアントを初期化するコア ロジックを処理します。 グループ参加をサポートします。 | React | N/A     | N/A |
| メディア コントロール   | ミュートの切り替え、ビデオのオンとオフの切り替え、通話の終了を行うことにより、ユーザーが現在の通話を管理できるようにします。                                                                                                                                                              | React | N/A     | N/A |
| メディア ギャラリー   | 1 つのギャラリーで通話参加者全員を紹介します。 ギャラリーでは、ビデオ対応と静止画の両方で参加者を表示できます。 ビデオ対応の場合は、参加者のビデオがレンダリングされます。                                                                                                                | React | N/A     | N/A |
| マイク設定 | 通話に使用するマイクを選択します。 このコントロールを使用すると、通話の前と通話中にマイク デバイスを選択することが使用できます。                                                                                                                                               | React | N/A     | N/A |
| カメラ設定     | ビデオ通話に使用するカメラを選択します。 このコントロールを使用すると、通話の前と通話中にビデオ デバイスを選択することができます。                                                                                                                                             | React | N/A     | N/A |
| デバイス設定     | マイクとカメラの設定を 1 つのコンポーネントに結合します                                                                                                 | React | N/A     | N/A |
| チャット プロバイダー       | チャット用のコア初期化コンポーネント。 必須コンポーネント。このコンポーネントを基盤とする他のコンポーネントを初期化します。 Azure Communication Services のアクセス トークンと、チャット クライアントが参加しているスレッドを使用して、チャット クライアントを初期化するコア ロジックを処理します。                                     | React | N/A     | N/A |
| 送信ボックス          | ユーザーがチャット スレッドにメッセージを送信できるようにする入力コンポーネント。 入力は、テキスト、ハイパーリンク、絵文字などの Unicode 文字 (他のアルファベットを含む) をサポートします。                                                                                                                         | React | N/A     | N/A |
| チャット スレッド           | 送信者情報と共に、受信メッセージと送信メッセージの両方をユーザーに示すスレッド コンポーネント。 このスレッドは、入力インジケーターと開封確認メッセージをサポートします。 これらのスレッドをスクロールして、チャット履歴を確認できます。
| 参加者リスト      | 通話スレッドまたはチャット スレッドのすべての参加者を一覧として表示します。  | React | N/A     | N/A |
