# <a name="samples"></a>サンプル

[!INCLUDE [Public Preview Notice](../includes/public-preview-include-android-ios.md)]

Azure Communication Services には多くのサンプルが用意されており、これらは独自のアプリケーションやユース ケースを作成する前に ACS のサービスと機能をテストするために使用できます。

## <a name="application-samples"></a>アプリケーション サンプル

| サンプル名                                                                                          | 説明                                                         | 使用可能な言語/プラットフォーム      |
| :--------------------------------------------------------------------------------------------------- | :------------------------------------------------------------------ | :--------------------------------- |
| [グループ通話のヒーロー サンプル](./calling-hero-sample.md)                                                | グループ呼び出しアプリケーションを作成する例を示します。          | [Web](https://github.com/Azure-Samples/communication-services-web-calling-hero)、[iOS](https://github.com/Azure-Samples/communication-services-ios-calling-hero)、[Android](https://github.com/Azure-Samples/communication-services-android-calling-hero)                           |
| [Web 呼び出しのサンプル](./web-calling-sample.md)                                                        | Web 内の ACS 呼び出し機能に関するステップ バイ ステップのチュートリアルです。 | [Web](https://github.com/Azure-Samples/communication-services-web-calling-tutorial/)                                |
| [チャットのヒーロー サンプル](./chat-hero-sample.md)                                                            | チャット アプリケーションを作成する例を示します。                   | [Web](https://github.com/Azure-Samples/communication-services-web-chat-hero)                    |
| [Contoso 医療アプリ](https://github.com/Azure-Samples/communication-services-contoso-med-app)       | 患者と医師のフローを示すサンプル アプリ。                     | Web および Node.js                      |
| [Contoso リテール アプリ](https://github.com/Azure-Samples/communication-services-contoso-retail-app)     | 小売業のサポート フローを説明するサンプル アプリ。                     | ASP.NET、.NET Core、JavaScript/Web |
| [WPF 呼び出しのサンプル](https://github.com/Azure-Samples/communication-services-web-calling-wpf-sample) | 呼び出し機能をデモンストレーションする Windows 用のサンプル アプリ          | WPF / Node.js                      |
