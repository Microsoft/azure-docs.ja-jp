# <a name="switching-organizations-in-a-portal"></a>ポータルでの組織の切り替え

自身のアカウントまたはアクセスポータル (組織のアカウントの **マイ アカウント** ポータルなど) では、連携する他の組織に切り替えて、その組織でアカウント情報を管理することができます。

以下のポータルで、組織を切り替えることができます。

- [**マイ アカウント**](https://myaccount.microsoft.com)
- [**マイ アプリ**](https://myapps.microsoft.com)
- [**マイ アクセス**](https://myaccess.microsoft.com)
- [**マイ スタッフ**](https://mystaff.microsoft.com)

**自分のサインイン**、**セキュリティ情報**、または **マイグループ** では、組織を切り替えることはできません。

## <a name="switch-organizations"></a>組織の切り替え

別の組織に切り替える方法を次に示します。 この例では **マイ アカウント** ポータルを使用していますが、他のポータルでもスイッチャー アイコンとその位置は似ています。

1. 職場または学校アカウントにサインインした後、[[マイ アカウント]](https://myaccount.microsoft.com) ページを開きます。
1. ページ上部のアカウント写真の横にある組織スイッチャー :::image type="content" source="media/cross-tenant-switcher/switcher-icon.png" alt-text="組織切り替え用アイコン"::: を選択します。

    :::image type="content" source="media/cross-tenant-switcher/switcher-location.png" alt-text="切り替え先として選択可能な組織のメニュー":::

1. 切り替え先の組織を選択します。 選択した組織に自動的に切り替えられます。

選択した組織に追加のセキュリティ要件がある場合は、追加のセキュリティ情報を入力するように求められることがあります。

## <a name="after-switching"></a>切り替え後

組織の切り替え後、すべてのポータル ページには、選択した組織が反映されます。

> [!Important]
>ホーム組織へのサインインに使用するパスワードは、他の組織へのサインインに使用するパスワードです。 次の点に注意してください。
>
>- ホーム組織ではない組織にサインインしていて、 **[マイ アカウント]** の [概要] ページまたは **[パスワード]** が表示されるその他の場所でパスワードを選択した場合は、すぐにホーム組織に再度切り替えられます。
>- **マイ スタッフ** ポータルで別の組織に切り替えた場合は、マイ スタッフにのみアクセスできます (その組織内のマイ スタッフに対して有効になっている場合)。
