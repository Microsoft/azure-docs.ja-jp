# <a name="my-profile-preview-portal-overview"></a>マイ プロファイル (プレビュー) ポータルの概要

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-preview-notice-myprofile.md)]

**マイ プロファイル** (プレビュー) ポータルを使用して、職場または学校アカウントの次のような管理を行うことができます。

- 2 段階認証とパスワードのリセットに関するセキュリティ情報を設定および管理します。

- 接続されている組織およびデバイスを管理します。

- 組織でのデータの使用方法を表示します。

>[!Important]
>このコンテンツはユーザーを対象としています。 管理者の方は、Azure Active Directory (Azure AD) 環境の設定と管理方法の詳細については、「[Azure Active Directory のドキュメント](https://docs.microsoft.com/azure/active-directory)」を参照してください。

## <a name="access-the-my-profile-portal"></a>マイ プロファイル ポータルにアクセスする
次のいずれかのブラウザーの最新バージョンで https://myprofile.microsoft.com にアクセスして、**マイ プロファイル** ポータルに移動できます。

- Chrome
- Microsoft Edge
- Safari
- Firefox
- Internet Explorer 11

    ![マイ プロファイル ポータル、概要ページ](media/myprofile-portal/myprofile-portal-overview.png)
