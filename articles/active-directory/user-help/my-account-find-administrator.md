# <a name="find-an-administrator-to-change-my-information-in-my-account"></a>マイ アカウントの自分の情報を変更する管理者を見つける

IT や人事部門は、プロフィール情報の一部 (たとえば、プロフィールの写真、名前、職名など) を別のシステムに保存する場合があります。 この情報を自分で変更することが許可されていない場合は、人事部門または管理者に連絡して、変更するように依頼します。

## <a name="why-cant-you-just-give-me-the-name-of-who-to-contact"></a>連絡する担当者の名前を教えてもらえないのはなぜですか?

連絡する担当者の名前やメール アドレスをお知らせしたいのですが、Microsoft はこの情報を所有していません。 この記事では、自分の管理者を見つけるのに役立つ提案をいくつか示します。

## <a name="how-do-i-find-out-who-my-admin-is"></a>自分の管理者を見つけるにはどうすればよいですか?

特定の情報の更新、パスワードのリセット、アカウントの削除、その他のタスクを実行するためにマイ アカウントの管理者を探す場合に、誰に連絡を取るべきかについてのいくつかのポイントをここに示します。

- 大学と学校: テクニカル サポート チームに連絡します。 通常、大学のサイトでリンクを見つけることができます。 小規模な学校では、管理者のアクセス許可を持つ技術担当者が数名しかいない場合があります。

- 大企業: 組織内のヘルプ デスクまたはテクニカル サポートに連絡します。

- 小規模企業: 経営者または共同経営者に連絡します。 多くの場合、会社のコンピューター メンテナンス業務をすべて行う IT コンサルタントに管理者のアクセス許可が付与されています。

職場または学校で誰に支援を求めればよいかわからない場合は、マイ アカウントのユーザー アカウントとパスワードを付与した担当者に連絡してみてください。
