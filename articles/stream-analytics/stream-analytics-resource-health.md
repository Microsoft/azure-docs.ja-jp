# <a name="stream-analytics-resource-health-blade"></a>Stream Analytics の [リソース正常性] ブレード

Azure Portal の **[リソース正常性]** ブレードには、Azure Stream Analytics ジョブの正常性に関する情報が表示されます。 ここには、問題のトラブルシューティングを行ううえで役に立つ実用的なガイダンスも表示されます。 

**[リソース正常性]** ブレードは、問題の原因がアプリケーションにあるかどうか、または原因が Azure のイベントにあるかどうかを、より迅速に判断できるようにするためのものです。 これは、サポート ケースを開いてサポートを受けるタイミングを判断するのに役立ちます。

## <a name="troubleshooting-workflow"></a>トラブルシューティングのワークフロー

Stream Analytics ジョブのリソース正常性を確認するには、**[リソース正常性]** ブレードを選択します。 

![Stream Analytics のトラブルシューティング フロー - アクセス](media/stream-analytics-resource-health/stream-analytics-access-troubleshooting.png)

次のフローチャートは、調査対象とサポート ケースを開くタイミングを判断するのに役立ちます。

![Stream Analytics のトラブルシューティング フロー - マップ](media/stream-analytics-resource-health/stream-analytics-troubleshooting-map.png)
