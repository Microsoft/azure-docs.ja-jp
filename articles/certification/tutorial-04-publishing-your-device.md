# <a name="tutorial-publish-your-device"></a>チュートリアル: デバイスを発行する

おめでとうございます。無事、デバイスは認定されました。 今後は、Azure で適切に動作する優れたデバイスのエコシステムに、貴社の製品も加わることとなります。 認定されたデバイスを世界中の顧客が発見し、購入できるよう、ご希望であれば [Azure Certified Device カタログ](https://devicecatalog.azure.com)にデバイスの詳細を発行することができます。

このチュートリアルでは、以下の内容を学習します。

> [!div class="checklist"]
> * Azure Certified Device カタログにデバイスを発行する

## <a name="prerequisites"></a>前提条件

- [Azure Certified Device ポータル](https://certify.azure.com)にサインイン済みであり、デバイス用の **承認** されたプロジェクトがある必要があります。 認定デバイスがない場合は、こちらの[チュートリアル](tutorial-01-creating-your-project.md)を参照して開始できます。

## <a name="publishing-your-device"></a>デバイスを発行する

デバイスの発行は、Azure Certified Device カタログによって貴社の製品の集客を支援する単純なプロセスです。

1. デバイスを発行するには、プロジェクト サマリー ページの [`Publish to Device Catalog`]\(デバイス カタログへの発行\) をクリックします。

    ![カタログに発行する](./media/images/publish-to-catalog.png)

1. ポップアップ ウィンドウで発行を確認する

    ![カタログへの発行の確認](./media/images/publish-to-catalog-confirm.png)

1. Azure Certified Device カタログに対するデバイスの処理が完了すると、企業情報にあるメール アドレスに通知が届きます。

## <a name="next-steps"></a>次のステップ

お疲れさまでした。 認定デバイスは Azure Certified Device カタログに登録されました。顧客は、Azure でのそのパフォーマンスを信頼して貴社の製品を購入することができます。 認定 IoT 製品のエコシステムにご参加くださり、ありがとうございます。 プロジェクト ページが読み取り専用になったことにお気付きでしょうか。 デバイス情報の更新を希望される場合は、ハウツー ガイドを参照してください。
