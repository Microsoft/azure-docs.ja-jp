# <a name="azure-ad-domain-services---contact-us"></a>Azure AD ドメイン サービス - お問い合わせ

## <a name="feedback"></a>フィードバック
サービスに対するフィードバックや技術的でない質問がある場合、電子メールで Azure AD Domain Service 製品グループにご要望をお寄せいただけます。[Azure AD Domain Services のフィードバック](mailto:aaddsfb@microsoft.com)。 このアドレスへの電子メールはレビューの対象です。 さらに情報が必要な場合、製品グループのメンバーが返信する**ことがあります**。 この電子メール アドレスにテクニカル サポートを依頼されましても回答いたしかねますのでご了承ください。

## <a name="technical-assistance"></a>技術的支援
マネージド ドメインに問題がある場合は、[トラブルシューティング ガイド](troubleshoot.md)で説明されている手順で問題が解決するかどうかを確認してください。 問題が解決しない場合、Azure portal にサインインします。 **[Azure AD Domain Services]** ホーム ページを開き、 **[サポート + トラブルシューティング]** の下にあるアクション ウィンドウから **[新しいサポート要求]** を選択します。

![new support request](./media/contact-us/supportRequest.png) 

サポート要求の各セクションはできるだけ正確に入力してください。 サービスを選択するときは、必ず **[Azure Active Directory Domain Services (VM - Domain Controllers)]\(Azure Active Directory Domain Services (VM - ドメイン コントローラー)\)** を選択して、要求が適切なサポート担当者に届くようにしてください。

> [!IMPORTANT]
> Azure Active Directory でのドメイン検証の支援が必要な場合、サービスの一覧から **[Azure Active Directory Directories, Domains, and Objects]\(Azure Active Directory ディレクトリ、ドメイン、およびオブジェクト\)** を選択します。
>
> 

## <a name="feature-requests"></a>機能のリクエスト
Azure Active Directory Domain Services を改善する方法についてのアイデアをお持ちですか。 ぜひお聞かせください。 Azure フィードバック フォーラムにアクセスし、[[Domain Services]](https://feedback.azure.com/forums/169401-azure-active-directory?category_id=160593) カテゴリに移動して、アイデアを共有してください。 製品グループのメンバーがこれらの要求を確認します。
