# <a name="how-to-manage-pricing-tier-of-your-azure-maps-account"></a>Azure Maps アカウントの価格レベルを管理する方法

Azure Maps アカウントの価格レベルは、Azure portal で管理することができます。 [アカウント](https://azure.microsoft.com/free/?WT.mc_id=A261C142F)を作成すると、アカウントの価格レベルを表示したり、変更したりできるようになります。

適切な価格レベルの選択について詳しくは、[適切な価格レベルの選択](https://docs.microsoft.com/azure/azure-maps/choose-pricing-tier)に関する記事をご覧ください

## <a name="view-pricing-tier"></a>価格レベルの表示

選択した価格レベルを表示するには、設定メニューの **[価格レベル]** オプションに移動します。

![選択した価格レベルの表示](./media/how-to-manage-pricing-tier/view-pricing-tier.png)

## <a name="change-pricing-tier"></a>価格レベルの変更

Azure Maps アカウントを作成すると、Azure Maps アカウントの価格レベルをアップグレードしたり、ダウングレードしたりできるようになります。 アップグレードやダウングレードを行うには、設定メニューの **[価格レベル]** オプションに移動し、強調表示されていない価格レベルを選択します。 保存ボタンをクリックして、選択した価格レベル オプションを保存します。

![価格レベルの変更](./media/how-to-manage-pricing-tier/change-pricing-tier.png)

> [!Note]
> Azure Maps アカウントの価格レベルをアップグレードまたはダウングレードした場合、新しいサブスクリプション キーを生成する必要はありません。
