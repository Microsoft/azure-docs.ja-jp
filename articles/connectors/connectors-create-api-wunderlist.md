# <a name="monitor-and-manage-wunderlist-by-using-azure-logic-apps"></a>Azure Logic Apps を使用して Wunderlist の監視と管理を行う

2020 年 5 月 6 日以降、[Wunderlist サービスがシャットダウンされ](https://www.wunderlist.com/)、[データを移行して、代わりに Microsoft To Do の使用を開始](https://www.wunderlist.com/blog/goodbye-from-wunderlist/)するよう顧客に求めています。 [Microsoft To Do コネクタ](/connectors/todo/)を使用して、非推奨の [Wunderlist コネクタ](/connectors/wunderlist/)を置き換えることができます。

## <a name="next-steps"></a>次のステップ
