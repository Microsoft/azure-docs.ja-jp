# <a name="live-events-and-live-outputs"></a>ライブ イベントとライブ出力

Azure Media Services では、Azure クラウドで顧客にライブ イベントを配信することができます。 Media Services v3 でライブ ストリーミング イベントを構成するには、[ライブ イベント](https://docs.microsoft.com/rest/api/media/liveevents)と[ライブ出力](https://docs.microsoft.com/rest/api/media/liveoutputs)について理解する必要があります。
