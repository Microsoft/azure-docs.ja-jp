<!--Create a media services asset with Postman-->

このセクションでは、Postman を使用して新しいアセットを作成する方法を示します。

1. **[Assets]\(アセット\)**  ->  **[Create or update an Asset]\(アセットを作成または更新する\)** を選択します。
2. **[送信]** をクリックします。 新しく作成されたアセットに関する情報を含む**応答**が表示されます。
