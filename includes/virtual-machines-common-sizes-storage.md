
Ls シリーズは、NoSQL データベース (Cassandra、MongoDB、Cloudera、Redis など) のように、待ち時間の短いローカル ストレージを必要とするワークロードに最適化されています。 Ls シリーズでは、[Intel® Xeon® プロセッサ E5 v3 ファミリ](http://www.intel.com/content/www/us/en/processors/xeon/xeon-e5-solutions.html)を使用し、最大 32 個の CPU コアを提供します。 これは、G/GS シリーズと同じ CPU パフォーマンスであり、CPU コアあたり 8 GiB のメモリを搭載しています。
