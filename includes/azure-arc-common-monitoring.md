- **[Azure Data Studio ダッシュボードで監視する](../articles/azure-arc/data/azure-data-studio-dashboards.md)**
> Azure Data Studio とは、Windows、macOS、Linux 上でオンプレミス プラットフォーム、クラウド データ プラットフォームの Microsoft ファミリを使用するデータ プロフェッショナルを対象にした、クロスプラットフォーム データベース ツールです。 Azure Data Studio では、IntelliSense、コード スニペット、ソース管理の統合、統合されたターミナルを含む最新のエディター エクスペリエンスが提供されています。 これは、データ プラットフォームのユーザーを念頭に置いて設計されており、クエリ結果セットのグラフ化機能とカスタマイズ可能なダッシュボードが組み込まれています。

**[Grafana で監視する](../articles/azure-arc/data/monitor-grafana-kibana.md)**
> Grafana は、マルチプラットフォームでオープン ソースの、分析と対話型視覚化のための Web アプリケーションです。 サポートされているデータ ソースに接続しているときに、Web にチャート、グラフ、およびアラートを提供します。 プラグイン システムを通じて拡張できます。 エンド ユーザーは、対話型クエリを使用して、複雑な監視ダッシュボードを作成できます。

**[Kibana を使用したログ分析](../articles/azure-arc/data/monitor-grafana-kibana.md)**
>Kibana は、オープン ソースの、Elasticsearch 用のデータ視覚化ダッシュボードです。 Elasticsearch クラスターでインデックスが作成されたコンテンツに対する視覚化機能を提供します。 ユーザーは、大量のデータに関する横棒グラフ、折れ線グラフ、散布図、または円グラフと、マップを作成できます。
