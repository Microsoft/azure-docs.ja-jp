> [!TIP]
> 既定では、関数アプリに必要な Azure リソースが、指定した関数アプリ名に基づいて作成されます。 また、既定では、関数アプリを含んだ同じ新しいリソース グループがその作成先となります。 それらのリソースの名前をカスタマイズしたり、既存のリソースを再利用したりする場合は、[高度な作成オプションを使用してプロジェクトを発行](../articles/azure-functions/functions-develop-vs-code.md#enable-publishing-with-advanced-create-options)する必要があります。
