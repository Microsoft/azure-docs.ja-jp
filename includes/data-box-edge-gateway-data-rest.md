保存データの場合:

- 共有に格納されたデータへのアクセスは制限されます。

    - 共有データにアクセスする SMB クライアントには、その共有に関連付けられたユーザー資格情報が必要です。 これらの資格情報は、その共有が作成されるときに定義されます。
    - 共有が作成されるときは、その共有にアクセスする NFS クライアントの IP アドレスを追加する必要があります。
