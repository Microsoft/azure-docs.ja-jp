> [!NOTE]
>
> この記事は、新しい Azure PowerShell Az モジュールのコマンドを使用するように最近更新されました。 このモジュールは、Azure 用の新しい PowerShell モジュールであり、PowerShell バージョン 5.x および 6 と互換性があります。 AzureRM では重要なバグ修正の更新が継続されますが、新機能は Az モジュールだけに追加されます。
>
> * Az モジュールと AzureRM モジュールに対する継続的なサポートについて詳しくは、「[Introducing the new Azure PowerShell Az module (新しい Azure PowerShell Az モジュールの導入)](/powershell/azure/new-azureps-module-az)」をご覧ください。
> * Az モジュールは Cloud Shell 上で既に使用可能であり、AzureRM との互換性が自動的に有効になっています。
>   Az モジュールのインストール手順については、[Azure PowerShell のインストール](/powershell/azure/install-az-ps)に関する記事をご覧ください。
> * AzureRM 互換性コマンドを有効にするには、インストール後に `Enable-AzureRmAlias` を実行します。
>   互換性について詳しくは、[Azure PowerShell AzureRM から Az への移行](/powershell/azure/migrate-from-azurerm-to-az)に関する記事をご覧ください。
