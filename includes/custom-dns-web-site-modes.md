カスタム ドメイン名は、**Shared** モード、**Basic** モード、**Standard** モードの Web Apps に限って設定できます。 Web アプリを **Free** モードから **Shared** モード、**Basic** モード、または **Standard** モードに切り替える場合、最初に Web Apps のサブスクリプションに設定されている使用制限を解除する必要があります。 

サイトのモードの変更方法など、App Service のプラン モードの詳細については、 [Web アプリのスケールの設定方法](../articles/app-service-web/web-sites-scale.md)に関するページを参照してください。
