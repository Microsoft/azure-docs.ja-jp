セキュリティ コントロールとは、セキュリティの脆弱性の防止、検出、および対処を行う Azure サービスの能力に寄与する、そのサービスの品質または機能です。

コントロールごとに "はい" または "いいえ" を使用して、それが現在サービスに適用されているかどうかを示し、サービスに該当しないコントロールについては "該当なし" を使用します。 また、属性に関するメモや詳細情報へのリンクが記載されている場合もあります。
