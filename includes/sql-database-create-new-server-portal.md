## <a name="create-an-azure-sql-database-logical-server"></a>Azure SQL Database 論理サーバーを作成する
次の手順に従い、Azure ポータルで Azure SQL Database 論理サーバーを作成します。

1. [Azure Portal](http://portal.azure.com)に接続していない場合は接続します。
2. **[新規]**、**[データ + ストレージ]** の順にクリックし、**[SQL Database]** を見つけます。
3. **[SQL Database]** をクリックして、[SQL Database] ブレードを開きます。
4. 次のサーバー プロパティの値を入力します。
   
   * サーバー名
   * サーバー管理者のログイン
   * パスワード
   * サブスクリプション (複数のサブスクリプションがある場合のみ)
   * リソース グループ (新規または既存)
   * 場所
5. **[作成]**をクリックします。 通知領域にデプロイの開始が表示されます。
6. デプロイが完了するまで待ってから、次の手順に進みます。
