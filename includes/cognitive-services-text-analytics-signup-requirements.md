Text Analytics リソースのキーとエンドポイント。 Azure Cognitive Services は、ユーザーがサブスクライブする Azure リソースによって表されます。 [Azure portal](../articles/cognitive-services/cognitive-services-apis-create-account.md) または [Azure CLI](../articles/cognitive-services/cognitive-services-apis-create-account-cli.md) を使用して、ローカル コンピューター上に Text Analytics のリソースを作成します。 リソースは、[Azure portal](https://portal.azure.com/) で表示することもできます
