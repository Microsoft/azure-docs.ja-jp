## <a name="get-started-with-azure-devtest-labs"></a>Azure DevTest Labs を使用する
Azure を利用するのが初めてであれば、 [無料の Azure アカウントを作成してください](https://azure.microsoft.com/free)。

既に Azure をご利用中であれば、 DevTest Labs で最初のラボの使用を開始します。
