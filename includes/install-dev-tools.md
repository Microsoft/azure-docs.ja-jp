> このセクションを適切に実行するには、正常に動作している Node.js のインストールと、ご利用のプラットフォーム用の Azure SDK for Node.js が必要です。
> 
> * Node.js 用の Azure SDK のインストール: [Windows インストーラー](https://go.microsoft.com/fwlink/?LinkId=254279) | [Mac インストーラー](https://go.microsoft.com/fwlink/?LinkId=253471) | [Linux ダウンロード](https://go.microsoft.com/fwlink/?LinkId=253472)
> * Mac または Linux を使用している場合は、[https://nodejs.org](https://nodejs.org) から Node.js をインストールします。Windows インストーラには既に Node.js が含まれています。
