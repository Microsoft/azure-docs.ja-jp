> [!IMPORTANT]
>
> PowerShell からこの Azure 機能を使用するには、`AzureRM` モジュールがインストールされている必要があります。 これは Windows PowerShell 5.x でのみ使用可能な古いモジュールで、重要な更新プログラムが提供されますが、新しい機能は今後追加されません。 `Az` モジュールがインストールされている場合は、`AzureRM` もインストールする必要があります。 `AzureRM` をインストールするには:
> 
> 1. `Disable-AzureRMAlias` を使用して `Az` の AzureRM の別名を無効にします (有効になっている場合)。 `Az` がインストールされていない状態または別名が無効な状態でこのコマンドを実行しても問題はありません。
> 2. [AzureRM の最新のインストール手順](/powershell/azure/azurerm/install-azurerm-ps)に従います。 `AzureRM` がインストールされている場合でも、最新バージョンに更新します。
