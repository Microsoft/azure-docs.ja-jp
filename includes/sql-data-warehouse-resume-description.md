データベースを再開する場合

* SQL Data Warehouse では、DWU 設定のコンピューティング リソースとメモリ リソースを取得します。
* DWU の再開に対する料金を計算します。
* データが使用可能になります。
* ワークロード クエリを再実行する必要があります。
