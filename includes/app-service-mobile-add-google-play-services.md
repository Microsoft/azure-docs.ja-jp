1. Android Studio のツール バーのアイコンをクリックするか、またはメニューから **[Tools (ツール)]** > **[Android]** > **[SDK Manager (SDK マネージャー)]** を順にクリックして、Android SDK マネージャーを開きます。 **[Launch Standalone SDK Manager (スタンドアロン SDK マネージャーの起動)]** リンクをクリックします。
2. プロジェクトで使用するバージョンの Android SDK を見つけて開きます。 **Google API** がインストールされていない場合は、**[Install packages... (パッケージのインストール)]** をクリックします。
3. Android Studio で、**[File (ファイル)]** > **[Project Structure (プロジェクト構造)]** の順に選択します。 **[Notifications (通知)]** を選択し、**[Google Cloud Messaging]** を選択して **[OK]** をクリックします。
