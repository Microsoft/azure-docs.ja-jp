Azure CloudShell を使用している場合は、アカウントに自動的に接続されるため、Connect-AzAccount コマンドを実行する必要はありません。 PowerShell をローカルで実行している場合は、昇格された特権で PowerShell コンソールを開き、`Connect-AzAccount` コマンドを使用して Azure に接続します。
