次の制限が Azure Site Recovery に適用されます。

| 制限の種類 | 制限 |
| --- | --- |
| サブスクリプションあたりの資格情報コンテナーの数 |500 |     
| Recovery Services コンテナーあたりのサーバーの数 |250 |
| Recovery Services コンテナーあたりの保護グループの数 |制限なし |
| Recovery Services コンテナーあたりの復旧計画の数 |制限なし |
| 保護グループあたりのサーバーの数 |制限なし |
| 復旧計画あたりのサーバーの数 |100 |
