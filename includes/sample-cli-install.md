このサンプルを実行するには、最新バージョンの [Azure CLI](https://docs.microsoft.com/cli/azure/install-azure-cli) がインストールされていることを確認してください。 開始するには、`az login` を実行して、Azure との接続を作成します。

このサンプルは、bash シェルに対応しています。 Azure CLI スクリプトを Windows クライアントで実行する方法については、「[Windows での Azure CLI 2.0 のインストール](/cli/azure/install-azure-cli-windows)」をご覧ください。
