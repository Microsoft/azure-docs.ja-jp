デバイス上のリソースの消費に関連するアラート条件を通知するようにアラート ルールを構成します。 アラート条件についてデバイスを監視するようにアラート ルールを構成できます。 アラートの詳細については、「[Create, view, and manage metric alerts in Azure monitor](../articles/azure-monitor/platform/alerts-metric.md)」(Azure Monitor でメトリック アラートを作成、表示、管理する) を参照してください。
