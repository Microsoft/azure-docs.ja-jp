> ジョブが実行されていない場合の課金を回避するには、**最小ノード数を 0 に設定** します。 この設定により、使用されていないノードの割り当てを Azure Machine Learning で解除できます。 0 より大きい値を指定すると、ノードが使用されていない場合でも、指定した数のノードが実行され続けます。
