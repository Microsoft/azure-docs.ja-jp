Azure Spatial Anchors SDK は、次のパッケージを使用して <a href="https://unity.com/releases/2019-lts" target="_blank">Unity 2019.4 (LTS)</a> で構成する必要があります。
- AR Foundation:<a href="https://docs.unity3d.com/Packages/com.unity.xr.arfoundation@3.1/manual/index.html" target="_blank">3.1.3</a>
- Windows Mixed Reality:<a href="https://docs.unity3d.com/Packages/com.unity.xr.windowsmr.metro@4.2/manual/index.html" target="_blank">4.2.1</a>
- ARCore XR プラグイン (Android デバイス サポート用):<a href="https://docs.unity3d.com/Packages/com.unity.xr.arcore@3.1/manual/index.html" target="_blank">3.1.3</a>
- ARKit XR プラグイン (iOS デバイス サポート用):<a href="https://docs.unity3d.com/Packages/com.unity.xr.arkit@3.1/manual/index.html" target="_blank">3.1.3</a>

> [!NOTE]
> ASA SDK は <a href="https://docs.unity3d.com/Manual/XRPluginArchitecture.html" target="_blank">Unity XR プラグイン フレームワークを現在サポートしていません</a>
