バージョン2.9 以降の Azure Spatial Anchors SDK は、次のパッケージを使用して[UNITY XR プラグイン フレームワーク](https://docs.unity3d.com/Manual/XRPluginArchitecture.html)で[UNITY 2020.3 (LTS)](https://unity3d.com/unity/whats-new/2020.3.0)において構成する必要があります。

- AR Foundation: [4.0.12](https://docs.unity3d.com/Packages/com.unity.xr.arfoundation@4.0/manual/index.html)
- Windows XR プラグイン (HoloLens デバイス サポート用): [4.4.1](https://docs.unity3d.com/Packages/com.unity.xr.windowsmr@4.4/manual/index.html)
- ARCore XR プラグイン (Android デバイス サポート用): [4.0.12](https://docs.unity3d.com/Packages/com.unity.xr.arcore@4.0/manual/index.html)
- ARKit XR プラグイン (iOS デバイス サポート用): [4.0.12](https://docs.unity3d.com/Packages/com.unity.xr.arkit@4.0/manual/index.html)

Azure Spatial Anchors SDK バージョン2.8 以前では、 [UNITY XR プラグイン フレームワーク](https://docs.unity3d.com/Manual/XRPluginArchitecture.html) はサポートされていないため、次のパッケージを使用して [UNITY 2019.4 (LTS)](https://unity.com/releases/2019-lts) で構成する必要があります。

- AR Foundation:[3.1.3](https://docs.unity3d.com/Packages/com.unity.xr.arfoundation@3.1/manual/index.html)
- Windows Mixed Reality (HoloLens デバイス サポート用): [4.2.1](https://docs.unity3d.com/Packages/com.unity.xr.windowsmr.metro@4.2/manual/index.html)
- ARCore XR プラグイン (Android デバイス サポート用):[3.1.3](https://docs.unity3d.com/Packages/com.unity.xr.arcore@3.1/manual/index.html)
- ARKit XR プラグイン (iOS デバイス サポート用):[3.1.3](https://docs.unity3d.com/Packages/com.unity.xr.arkit@3.1/manual/index.html)
