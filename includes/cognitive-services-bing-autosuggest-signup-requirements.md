Bing Autosuggest API にアクセスできる [Cognitive Services API アカウント](https://docs.microsoft.com/azure/cognitive-services/cognitive-services-apis-create-account)を所有している必要があります。 Azure サブスクリプションをお持ちでない場合は、無料で[アカウントを作成](https://azure.microsoft.com/try/cognitive-services/?api=bing-news-search-api)できます。 無料試用版をアクティブ化した後に提供されるアクセス キー、または Azure ダッシュボードから入手できる有料のサブスクリプション キーがあらかじめ必要となります。
